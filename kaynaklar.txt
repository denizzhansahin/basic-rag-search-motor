PROGRAM ÇALIŞTIRMA:
Oluşturulan tüm Docker konteynerlerini çalıştır, Docker Desktop ile yapabilirsin.

web_crawler klasörüne giriş yap, klasör içinde bunu çalıştır:
export PYTHONPATH=$PYTHONPATH:.
python site_manager.py


web_crawler klasörü içinde bunu çalıştır-düşük seviye işlemcin var ise:
python redis_queue.py

eğer güçlü bir işlemcin var ise ,  web_crawler içinde bunu çalıştır:
python redis_queue_worker.py

Admin panel ile siteleri ekleyebilirsin. Siteleri ise https:// veya http:// olarak ekle, ana alan adını eklemelisin, tercihi olarak her bir site için varsayılan arama sayfa sayısını değiştirmeyin:
web_crawler içinde bunu çalıştır:
export STREAMLIT_WATCHER_TYPE=none
streamlit run admin_panel.py
Örnek site ekleme : https://www.denizhansahin.com 

Arama motoruna ulaşmak için, search_web içinde bunu çalıştır:
export STREAMLIT_WATCHER_TYPE=none
streamlit run main_app.py


Eğer sohbet robotuna ulaşmak istersen:
ollama_run klasörü içine gir:
export STREAMLIT_WATCHER_TYPE=none
streamlit run app.py
