Docker ayağa kaldır:
docker-compose up --build




Docker konteynarlarını kapatmak için:
docker-compose down


"Hayalet" Ağları Temizlemek istersen:
Docker'ın belleğinde kalan ve kullanılmayan ağları şu komutla zorla temizleyebilirsin:
docker network prune -f


İsteğe bağlı python_worker sayısını arttırabilirsin:
docker-compose up --scale nestjs_gateway=2 --scale python_worker=3 -d

İsteğe bağlı python_worker sayısını arttırabilirsin:
docker-compose up --scale python_worker=3






TÜM DOCKER SIfIRLAMA: DİKKATLİ OLUNUZ:
Eğer sorun hala devam ederse, Docker Desktop'ı (üst menüdeki balina simgesinden) tamamen kapatıp yeniden açmak veya şu komutla her şeyi (imajlar hariç) sıfırlamak işe yarayabilir:
docker system prune
